# x-spaces-api

Get Spaces information for users on X (ex-Twitter)

## Features

- TypeScript
- Wrangler to develop and deploy.

## Setup

### Install

```shell
yarn install
```

### Configuation

```shell
npx wrangler login
```

### Develop

```shell
yarn dev
```

### Test

```shell
yarn test
```

### Deploy

```shell
yarn deploy
```
